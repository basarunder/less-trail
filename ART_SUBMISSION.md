MEANING IS A GAMBLE: A Generative Syntax Experiment
Artist: Başar Ünder

Year: 2025
Medium: Web-based Generative Software (p5.js, Datamuse API, Custom ASCII algorithms)

Concept

This installation is a digital study in chance, constraint, and meaning. It strips the analog act of rolling dice of its numerical value and transforms it into a lexical event. Ultimately, meaning is a gamble.

When you interact with the system, two virtual dice are cast. Their values do not represent a score. Instead, they dictate the strict structural limits of a new phrase by defining the exact length of two words. A roll of 3 and 6 does not equal 9. It creates a three-letter seed and a six-letter anchor (e.g., "EGO MANIAC").

The resulting texts are pulled from the Datamuse API. They range from the profoundly accurate to the completely absurd. Each combination creates a brief moment of sense before it is permanently wiped away for the next roll.

Rendered in an empty void using dot-matrix typography, the work positions the viewer as an active agent of chaos. You are invited to endlessly roll for meaning within a system governed by cold logic.

Technical Statement

The project is built on the p5.js creative coding framework and uses a custom rendering engine to display responsive ASCII graphics. The visual language relies on the "Doto" typeface to echo the heritage of early computing and terminal displays.

Beneath the visual layer, the system interfaces with the Datamuse API in real-time. It queries a linguistic database using the specific numerical constraints of the dice. The algorithm filters for basic semantic coherence while actively leaving room for surreal juxtapositions.

Viewing Instructions

Interact: Tap or click anywhere on the terminal screen to initiate the sequence.
Observe: Watch the probability collapse as the dice settle and the machine searches for a matching phrase.
Repeat: There is no win state. The cycle of generation is the artwork itself.

Keywords
